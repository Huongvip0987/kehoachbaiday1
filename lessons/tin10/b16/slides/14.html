VÍ DỤ 1 · KIỂU DỮ LIỆU

Python tự nhận biết kiểu dữ liệu

>>> 5
5
>>> 2.6
2.6
>>> "hoc sinh lop 10"
'hoc sinh lop 10'
Nhập vàoKiểuTên
5intSố nguyên
2.6floatSố thực
"abc"strXâu ký tự
TrueboolLôgic

💡 Quan sát

  • Nhập số nguyên → Python trả về số nguyên
  • Nhập số thực (có dấu .) → float
  • Chữ giữa "..." → xâu ký tự
  • Python tự nhận biết — không cần khai báo kiểu
