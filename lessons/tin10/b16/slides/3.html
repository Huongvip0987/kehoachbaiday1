Ba loại ngôn ngữ lập trình

Quan sát ba đoạn chương trình sau — cùng làm một việc: in kí tự "a" ra màn hình

⚙️ Ngôn ngữ máy

10110000 01100001
11001011 00000000

Gồm số 0 và 1 — máy tính mới hiểu trực tiếp

📋 Hợp ngữ (Assembly)

MOV AL, 61h
INT 21h

Dùng từ viết tắt tiếng Anh, vẫn còn phức tạp

🐍 Python (bậc cao)

print("a")

Gần với ngôn ngữ tự nhiên — dễ đọc, dễ hiểu
