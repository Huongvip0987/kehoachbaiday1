Chương trình dịch

Máy tính chỉ hiểu ngôn ngữ máy → cần chương trình dịch để chuyển từ NNLT bậc cao sang ngôn ngữ máy.

📝 Trình biên dịch (Compiler)

  • Dịch toàn bộ chương trình một lần
  • Tạo ra file thực thi (.exe)
  • Ví dụ: C, C++, Java
# Nguồn .c → dịch → .exe → chạy

⚡ Trình thông dịch (Interpreter)

  • Dịch và chạy từng dòng lệnh
  • Dễ kiểm tra, thử nghiệm nhanh
  • Python dùng cách này
# Dòng 1 → dịch → chạy → Dòng 2 → ...

🤔 Câu hỏi SGK: Viết chương trình bằng loại ngôn ngữ lập trình nào dễ nhất?  

→ C. Ngôn ngữ lập trình bậc cao — vì cú pháp gần với ngôn ngữ tự nhiên, không cần viết mã nhị phân hay từ viết tắt phức tạp.
