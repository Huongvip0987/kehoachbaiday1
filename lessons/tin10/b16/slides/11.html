CHẾ ĐỘ SOẠN THẢO

Chế độ soạn thảo chương trình

📁 Cách mở: File → New File (hoặc Ctrl+N) 
# Chuong trinh dau tien
print("Xin chao!")
print(5 + 2*7)
📝 Viết mỗi lệnh trên một dòng

📌 Các bước thực hiện

  • Bước 1: File → New File để mở soạn thảo
  • Bước 2: Nhập nội dung chương trình
  • Bước 3: Lưu tệp — Ctrl+S
  • Bước 4: Chạy — F5 (Run/Run Module)
  • Bước 5: Xem kết quả ở cửa sổ Python Shell
💡 Chú ý: Tệp Python có đuôi .py
