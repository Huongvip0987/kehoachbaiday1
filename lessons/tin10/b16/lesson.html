Chương trình dịch

Máy tính chỉ hiểu ngôn ngữ máy → cần chương trình dịch để chuyển từ NNLT bậc cao sang ngôn ngữ máy.

📝 Trình biên dịch (Compiler)

  • Dịch toàn bộ chương trình một lần
  • Tạo ra file thực thi (.exe)
  • Ví dụ: C, C++, Java
# Nguồn .c → dịch → .exe → chạy

⚡ Trình thông dịch (Interpreter)

  • Dịch và chạy từng dòng lệnh
  • Dễ kiểm tra, thử nghiệm nhanh
  • Python dùng cách này
# Dòng 1 → dịch → chạy → Dòng 2 → ...

  • Báo hiệu Python đang chờ nhận lệnh

Chế độ gõ lệnh trực tiếp

Gõ lệnh sau dấu nhắc >>> rồi nhấn Enter — Python thực hiện ngay lập tức.

CÂU HỎI CỦNG CỐ

Kiểm tra hiểu biết về Môi trường Python

🤔 Tự trả lời trước, rồi bấm 👁 để xem đáp án.

Câu 1: Dấu nhắc >>> là con trỏ soạn thảo chương trình Python. Đúng hay sai?

→ Sai — >>> là dấu nhắc trong chế độ gõ lệnh trực tiếp. Con trỏ soạn thảo xuất hiện trong cửa sổ soạn thảo (File → New File), không phải Shell.

Câu 2: Việc thực hiện câu lệnh ở hai chế độ có điểm gì giống và khác nhau?

→ Giống: đều dùng Python thực hiện, kết quả như nhau. Khác: trực tiếp — nhập lệnh → Enter → xem ngay; soạn thảo — viết nhiều lệnh, lưu file .py, nhấn F5 mới chạy.

💡 Quan sát

  • Nhập số nguyên → Python trả về số nguyên
  • Nhập số thực (có dấu .) → float
  • Chữ giữa "..." → xâu ký tự
  • Python tự nhận biết — không cần khai báo kiểu

  • Dùng // để lấy thương nguyên

VÍ DỤ 3 · LỆNH PRINT

Lệnh print()

>>> print(12)
12
>>> print(10, 3.4 + 4.1, "hoa binh")
10 7.5 hoa binh
>>> print("Day ba so chan:", 2, 4, 6)
Day ba so chan: 2 4 6
>>> print("3 + 7 =", 3 + 7)
3 + 7 = 10
📌 Ghi nhớ:   print(v1, v2, ..., vn) — in một hoặc nhiều giá trị, cách nhau dấu cách. Có thể tính toán biểu thức trong lệnh print.
CÚ PHÁP

Cú pháp lệnh print()

📌 Cú pháp

print(v1, v2, ..., vn)
  • v1, v2,..., vn là các giá trị cần in
  • Có thể là số, xâu ký tự, biểu thức
  • Các giá trị cách nhau dấu cách
  • Mỗi lệnh print() in một dòng riêng
# In một giá trị
print(42)

# In nhiều giá trị
print("Kết quả:", 10 + 5)

# In xâu và biểu thức
print("PI ≈", 3.14159)
print("1+2+3 =", 1+2+3)
CÂU HỎI CỦNG CỐ

Kiểm tra nhanh – SGK trang 88

🤔 Tính nhẩm trước, rồi bấm 👁 kiểm tra.

Câu 1a: Kết quả và kiểu dữ liệu của 5 / 2?

→ Kết quả: 2.5 — kiểu: float (phép / luôn cho kết quả thực)

Câu 1b: Kết quả của 12 + 1.5?

→ Kết quả: 13.5 — kiểu: float (int + float → float)

Câu 2: Lệnh print("13 + 10*3/2 - 3*2 =", 13 + 10*3/2 - 3*2) in ra gì?

→ 10×3=30, 30/2=15.0, 3×2=6 → 13+15.0−6 = 22.0
In ra: 13 + 10*3/2 - 3*2 = 22.0

VẬN DỤNG

Triple quotes và vòng lặp for

🔤 Xâu nhiều dòng – Triple quotes

print("""Khong co viec gi kho
Chi so long khong ben
Dao nui va lap bien
Quyet chi at lam nen""")

Cặp """...""" cho phép xâu chứa nhiều dòng — nhấn Enter giữa xâu.

🔄 Vòng lặp for – Bảng nhân 5

for i in range(1, 11):
    print("5 x", i, "=", 5*i)

range(1, 11) → i chạy từ 1 đến 10
Kết quả: 5×1=5, 5×2=10, ..., 5×10=50
