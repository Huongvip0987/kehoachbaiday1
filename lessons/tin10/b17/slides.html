KHỞI ĐỘNG

Biến trong toán học và lập trình là gì?

📐 Biến trong Toán học

Biến là ký hiệu đại diện cho một giá trị chưa xác định hoặc có thể thay đổi.

(a + b)² = a² + 2ab + b²
a, b đại diện cho bất kỳ số nào

→ Một công thức dùng được cho mọi bộ số a, b nhờ có biến.

💻 Biến trong Lập trình

Biến là tên của một ô nhớ trong máy tính dùng để lưu trữ dữ liệu. Giá trị có thể thay đổi trong quá trình chạy. 

diem = 8.5      # lưu số thực
ten  = "An"     # lưu xâu ký tự
tuoi = 16       # lưu số nguyên

→ Một chương trình dùng lại cùng tên biến nhiều lần, giá trị tự cập nhật.

🔑 Điểm giống nhau: Cả hai đều là ký hiệu thay thế cho giá trị — toán học dùng để biểu diễn công thức tổng quát, lập trình dùng để lưu và xử lý dữ liệu.

KIẾN THỨC MỚI

Gán nhiều biến = một giá trị · Toán tử kết hợp

# Gán nhiều biến cùng MỘT giá trị
a = b = 5          # Cả a và b đều bằng 5
a = b * 2          # a = 10 (vế phải tính trước!)
c = (a - b)**2     # c = (10-5)^2 = 25

🔄 Phân tích x = x + 1

  • Bước 1: Tính vế phải → x + 1
  • Bước 2: Gán kết quả vào x
  • Nên dù trông "vô nghĩa" về toán học, lệnh gán này hợp lệ!

⚡ Toán tử gán kết hợp

  • x += 1  ≡  x = x + 1
  • x -= 2  ≡  x = x - 2
  • x *= 3  ≡  x = x * 3
  • x //= 2 ≡  x = x // 2

VẬN DỤNG

Đổi giây ra ngày / giờ / phút / giây

Cho số giây ss = 684500. Viết chương trình đổi ra ngày, giờ, phút, giây.

🤔 Em hãy tự tính theo thuật toán bên trái, rồi bấm nút 👁 ở dòng kết quả để kiểm tra.

🧮 Thuật toán

  • ngay = ss // 86400
  • gio = (ss % 86400) // 3600
  • phut = (ss % 3600) // 60
  • giay = ss % 60

✅ Kết quả mẫu

  • 684500 giây = 7 ngày · 22 giờ · 8 phút · 20 giây
  • 1 ngày = 86400 giây
  • 1 giờ = 3600 giây
VẬN DỤNG

Tại sao Python không cần biến tạm để hoán đổi?

❌ Cách cũ (Pascal, C): 3 bước

x = 10; y = 7
# Cần biến tạm:
tam = x    # tam = 10
x = y      # x = 7
y = tam    # y = 10
print(x, y)  # 7 10

✅ Python: chỉ 1 dòng!

x, y = 10, 7
x, y = y, x   # Hoán đổi!
print(x, y)   # 7 10

# Bí mật: Python tạo tuple (y, x) = (7, 10)
# trước → rồi gán đồng thời vào x và y.
# Vế phải tính XONG trước khi gán!
🤔 Câu hỏi SGK: Trước lệnh x, y = y, x với x=10, y=7 thì x và y là bao nhiêu? Sau lệnh thì sao? Em có nhận xét gì về kết quả?
