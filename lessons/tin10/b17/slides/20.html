PHÉP TOÁN XÂU

Phép toán trên xâu ký tự

s1 = "Tin học"
s2 = " 10"

# + : nối chuỗi
print(s1 + s2)        # → Tin học 10

# * n : lặp chuỗi n lần
print("Ha! " * 3)     # → Ha! Ha! Ha!
print("=" * 20)        # → ====================

# Kết hợp + và * (dấu * tính trước!)
print("" * 9 + "Ok")           # → Ok (xâu rỗng lặp mấy lần vẫn rỗng)
print("5" + "0" * 4)           # → 50000
⚠️ Lưu ý: Không thể cộng số với xâu: x = 5 + "abc" → TypeError!
Muốn ghép số vào chuỗi, dùng str(x) + "abc" hoặc f"{x}abc"
