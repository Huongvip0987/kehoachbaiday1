TỔNG KẾT BÀI 17

Những gì đã học

📦 Biến

  • Tên của vùng nhớ lưu giá trị
  • Python tự xác định kiểu dữ liệu
  • Giá trị thay đổi được khi chạy

⬅ Lệnh Gán

  • Đơn: x = <biểu thức>
  • Đồng thời: x, y = 1, 2
  • Hoán đổi: x, y = y, x

🔢 Phép Toán

  • Số: + - * / // % **
  • Xâu: + (nối) * n (lặp)
  • Ưu tiên: ** > * / // % > + -

🔑 Từ Khóa

  • Từ đặc biệt của Python (35 từ)
  • Không được dùng làm tên biến
  • VS Code tô màu từ khóa tự động
