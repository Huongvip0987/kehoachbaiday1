Các tên biến sau có hợp lệ không? Giải thích.

TênHợp lệ?Giải thích
_ifHợp lệChỉ có if mới là từ khóa; thêm _ → hợp lệ
globalKhông hợp lệLà từ khóa Python (khai báo biến toàn cục)
nolocalHợp lệTừ khóa là nonlocal (khác chính tả)
returnKhông hợp lệLà từ khóa Python (trả về giá trị hàm)
trueHợp lệTừ khóa là True (T hoa); true thường ≠ từ khóa
