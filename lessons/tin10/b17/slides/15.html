Kiểu dữ liệu của biến trong Python

KiểuTên gọiVí dụ
intSố nguyêntuoi = 16
floatSố thựcdiem = 8.5
strXâu ký tựten = "An"
boolLôgicdung = True
💡 Hàm type(x) trả về kiểu dữ liệu hiện tại của biến x. 
x = 5
print(type(x))   # <class 'int'>

y = 3.14
print(type(y))   # <class 'float'>

s = "Python"
print(type(s))   # <class 'str'>

b = True
print(type(b))   # <class 'bool'>
⚠️ Lưu ý: Không thể thực hiện phép toán giữa các biến khác kiểu. Ví dụ: z = 5 + "abc" → TypeError! Phải đổi kiểu trước: str(5) + "abc"
