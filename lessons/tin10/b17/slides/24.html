🌍 Ứng dụng thực tế

  • Kiểm tra số chẵn: n % 2 == 0
  • Đổi giây → phút : giây
tong_giay = 135
phut = tong_giay // 60   # 2
giay = tong_giay % 60    # 15
print(phut, "phút", giay, "giây")
# → 2 phút 15 giây
