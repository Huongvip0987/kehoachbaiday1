SyntaxError – Lỗi khi dùng từ khóa làm tên biến

# Lỗi SyntaxError (SGK tr.95)
if = 12
# SyntaxError: invalid syntax

with = "Độ rộng"
# SyntaxError: invalid syntax

for = 5
# SyntaxError: invalid syntax
⚠️ Python báo SyntaxError ngay khi đọc code — trước khi chạy chương trình.

✅ Cách đặt tên thay thế

  • _if = 12  ← thêm dấu _
  • chieu_rong = "Độ rộng"  ← dịch nghĩa
  • for_loop = 5  ← thêm hậu tố
💡 VS Code tô màu xanh dương cho từ khóa — nếu tên biến cũng chuyển màu, đó là từ khóa!
