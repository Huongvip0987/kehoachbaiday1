MứcPhép toánHướng
1( ) Ngoặctrong → ngoài
2** Lũy thừa← phải → trái
3* / // %trái → phải
4+ -trái → phải

🧮 Giải ví dụ SGK tr.93

3/2 + 4*2**4 - 5//2**2
# B1: ** trước: 2**4=16; 2**2=4
# B2: Nhân/chia: 3/2=1.5; 4*16=64; 5//4=1
# B3: Cộng/trừ: 1.5+64-1 = 64.5

4**2**3   # Lũy thừa: phải → trái!

# Khác: (4**2)**3 = 16**3 = 4096
